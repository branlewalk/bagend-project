# bagend-project

### Hardware Requirements
  - RaspberryPi
  - Breadboard
  - 3 Temperature probes (DS18B20)
  - 1 Perma-Proto Half-sized Breadboard

### Installation

1. Download the repo
   ```sh
   git clone https://github.com/branlewalk/bagend-project.git
   ```
2. Build and Run Docker Image
   ```sh
   bash build-images.sh
